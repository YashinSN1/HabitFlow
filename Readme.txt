:: upcoming features ::

calendar = [pending]:

on mobile it will be a vertical list showing 30 past days and 7 upcoming days
on desktop it will be a month grid with a detail panel on the right side
clicking any date loads that day's habits from the backend
upcoming dates are locked and past dates can be edited
the window always generates fresh from today so nothing needs to be stored or updated manually

analytics and progress = [pending]:
current streak calculated on the backend
completion percentage over the last 7 30 and 90 days
a micro win display that shows small progress like doing something more than last month
an overall habit summary that connects small daily wins to the bigger goal
a feedback message per habit that changes based on whether you are struggling improving or consistent

ui and polish = [pending]:
mobile responsive layout for all pages
a priority system for habits with high medium and low levels
filter and sort habits by priority status or category
loading and error states on every page so nothing just breaks silently
dashboard cleanup with good empty states and consistent spacing
